# Robot Arm v. 0.1

RobotArm is a 3D simulation of a robotic arm. The model was build throught the Denavit-Hartenberg table.

## Dependence

* OpenGL
* Glut
* libxmu
* libxi

## How to build

### Linux

    mkdir build
    cd build
    cmake ..
    make
